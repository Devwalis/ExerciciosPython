Permissões de Arquivos no Linux

Estou aprendendo como funcionam permissões de leitura, escrita e execução no Linux.

Ver permissões
ls -l

Exemplo de saída:

-rw-r--r-- 1 walisson walisson 250 mai 13 funcoes.py
Tornar arquivo executável
chmod +x script.py
Executar arquivo diretamente
./script.py
Remover permissão de execução
chmod -x script.py
Git e Versionamento

Inicializando repositório:

git init

Verificando status:

git status

Adicionando arquivos:

git add .

Criando commit:

git commit -m "Adicionando exercícios Python"

Conectando ao GitHub:

git remote add origin URL_DO_REPOSITORIO

Enviando código:

git push -u origin main
